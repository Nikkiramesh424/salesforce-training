## What is Salesforce Platform?
Salesforce Platform is a cloud-based platform used to build and manage applications, business processes, and customer data. It allows companies to customize features, automate workflows, and create apps according to business needs.


## What is an App?

An App in Salesforce is a collection of tabs, objects, and features grouped together for a specific purpose.

Example:
Sales App, Service App, College Admission App.

## What is an Object?

An Object is used to store data in Salesforce. It is similar to a database table.

Example:
Account, Contact, Opportunity.

## What is a Tab?

A Tab is used to access objects, records, or pages in Salesforce UI.

Tabs help users navigate easily through the application.

## Difference Between Configuration and Coding

### Configuration

Configuration means creating features using clicks without writing code.

Examples:

- Creating fields
- Building flows

### Coding

Coding means writing custom logic using Apex or Lightning Web Components.

Examples:

- Complex validations
- Custom business logic


## Connecting Day 1 and Day 2

CRM concepts like Account, Contact, and Opportunity are stored as Objects in Salesforce.

These objects are organized inside Apps and accessed using Tabs.

This helps businesses manage customer data properly inside the Salesforce Platform.
